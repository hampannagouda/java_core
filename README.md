# Java Core Concepts Practice

This repository contains basic Java programs demonstrating fundamental Object-Oriented Programming (OOP) principles and core Java concepts.

## Files Included

### 1. Object-Oriented Programming (OOP) Concepts
* **`Single_level_Inheritance.java`**: Demonstrates the basic inheritance mechanism where a class extends one superclass.
* **`multilevel_inheritance.java`**: Shows a chain of inheritance where a class extends a child class, which in turn extends another parent class.
* **`hierarchical_inheritance.java`**: Illustrates a scenario where multiple child classes inherit from a single parent class.
* **`method_overriding.java`**: Demonstrates runtime polymorphism where a subclass provides a specific implementation of a method that is already provided by its parent class.

### 2. Core Java Fundamentals
* **`PrimitiveTypeCasting.java`**: Examples of widening (implicit) and narrowing (explicit) type casting between different primitive data types.
* **`PassByValue.java`**: Demonstrates how primitives are passed to methods by value.
* **`PassByReference.java`**: Shows how object references behave when passed to methods in Java.

## Usage
These programs are designed for educational purposes to understand the basics of Java programming. Each file is a standalone Java application with its own `main` method.
